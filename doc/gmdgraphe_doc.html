Spécifications GmdGraphe

Date : v 0.2 - 29/09/2006
Auteur : gm (gmarchault@yahoo.com)

Description

Représentation graphique dynamique d'un jeux de dataset dont les données sont issues d'un tableau.

Remarques sur l'utilisation de cette classe

- Php doit disposer du module gd compilé avec l'extension jpg et/ou png et/ou gif.
- Les images sont créées dans un répertoire tmp de la racine du serveur web : $DOCUMENT_ROOT/tmp
- Possibilité de définir un style css pour l'affichage du titre : gmdgraphe-titre
- Possibilité d'utiliser la classe dans le cadre d'un parsing de fichier texte, et stockage des informations dans un tableau pour les abscisses, et des valeurs correspondantes pour les ordonnées.

Utilisation

// Initialisation :
$mygraphe = new GmdGraphe( type_graphe, tableau_abscisse, tableau_ordonnees );

// Optionnel :
// ---------------
// Redéfinir les valeurs des abscisses
$mygraphe->set_abscisse( tableau_abscisses ); // Si tableau_abscisse == "MOIS", affichage des mois de l'année.
// Positionner des parametres (cf "Paramètres" ci-dessous)
$mygraphe->set_params( width, height, type_graphe, couleur, grid, bg_color, format, zero, aff_label, legende);
// Définir un nouveau jeux de données dans le meme graphe :
$mygraphe->get_graphe( type_graphe, type_graphe, tableau_abscisse, tableau_ordonnees );

// Affichage des données :
$mygraphe->set_graphe( titre ); // image graphique
// et / ou
$mygraphe->aff_data( cou ); // valeurs dans un tableau; paramètre cou=0 => pas de couleurs dans la légende.

Paramètres modifiables (optionnel) via la méthode set_params().
Si vous ne souhaitez pas modifier un paramètre , indiquez -999.

- width : largeur de l'image
- height : hauteur de l'image
- type_graphe : redefinir le type du graphe = { histogramme | point | ligne | air | camembert }
- couleur : tableau de couleurs prédefinies :

white, black,
gray, egray, lgray, dgray, slategray,
blue, lblue, dblue, marine, cyan, lcyan, steelblue, lsteelblue, skyblue, lskyblue, navy, slateblue,
yellow, lyellow, dyellow, lime, gold,
green, lgreen, dgreen, seagreen, lseagreen, khaki,
red, lred, dred, pink, dpink, purple, lpurple, dpurple, violet,
maroon, olive, tea, silver, tan, wheat

- grid : { true | false }, true par défaut.
- bg_color : couleur de fond de l'image ci-dessous.
- format : { jpg | png | gif }. Png par défaut.
- zero : { true | false } : Axe des x positionné sur 0, true par défaut.
- aff_label : { true | false } : Afficher ou non les intitulés des abscisses pour les camemberts.
- legende : tableau contenant les libellé des légendes pour chaque graphe, dans l'ordre des graphes tracés. Indiquez -999 si vous ne souhaitez pas de légende. Pas de légende par défaut.

Exemple :

$new_couleur = array ( "green", "blue", "red", "wheat", "seagreen", "cyan", "khaki" );
$t_abs = array ( "un", "deux", trois" ); $t_ord = array ( 32, 25, 64 );
$legende = array ( "Légende pour le tracé en histo");
$graphe1 = new GmdGraphe( "histogramme", $t_abs, $t_ord );

// Ajout d'un autre graphe, sous forme de ligne :
$val_moy = array (28, 28, 28);
$graphe1->get_graphe( "ligne", $t_abs, $val_moy );
array_push( $legende, "Moyenne" );

// Affichage
$graphe1->set_params( 600, 360, -999, $new_couleur, false, "white", -999, -999, -999, $legende );
$graphe1->set_graphe( "Exemple de graphiques" );

 Définition de style d'affichage:

Il est possible d'inclure les styles CSS suivants, correspondants respectivement (si besoin de les afficher) au titre du graphe, la légende, la liste des libellés des tables d'énumerationliés à un graphe :  .gmdgraphe-titre,  .gmdgraphe-legende,  .gmdgraphe-libelle-enum

Exemple de style CSS :

.gmdgraphe-titre { font-family:arial; font-size:11px; color:#0A1A58; font-weight:bold; }
.gmdgraphe-legende { font-family:arial; font-size:11px; color:#0A1A58; font-weight:bold; }
.gmdgraphe-libelle-enum { font-family:arial; font-size:11px; color:#C0C0C0; font-weight:bold; }
